# Skin Undertone Analyzer (Computer Vision Prototype)

An experimental computer vision project designed to classify skin undertones using HSV-based segmentation and heuristic color analysis.

## Overview
This project explores:
- Skin pixel isolation using HSV thresholding
- Noise reduction techniques
- Heuristic-based undertone classification
- Lighting robustness validation experiments

## Current Status
This project is under active refinement to improve:
- Classification consistency across diverse skin tones
- Calibration under different lighting environments
- Reduction of false positives in undertone detection

## Tech Stack
- Python
- OpenCV
- NumPy
- Streamlit

## Future Improvements
- Dataset-based validation (>= 50 labeled images)
- Confidence scoring system
- Statistical evaluation of undertone accuracy
